## Usage Tips

1. **Color Variations**: Each sprite sheet has 10 color rows - pick the one matching your game's element (fire=orange/red, ice=blue/cyan, nature=green, etc.)

2. **Animation Frames**: Read frames left-to-right for the animation sequence

3. **Transparency**: All sprites have transparent backgrounds (PNG format)

4. **Scaling**: These are pixel art sprites - scale in integer multiples (2x, 3x, 4x) for best results

5. **Looping**: Most effects are one-shot animations. For looping effects (like `fire_flame_burning_loop`), the animation seamlessly repeats.
